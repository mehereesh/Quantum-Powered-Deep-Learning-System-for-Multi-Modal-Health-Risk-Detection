About the Project

Brain Tumor Classification

This project leverages both Quantum Machine Learning (QML) and Classical Deep Learning (DL) models to classify brain MRI images and detect tumors. We used raw MRI image data for training, ensuring the model learns directly from authentic medical scans. The Pennylane library was utilized to simulate quantum circuits and extract features using quantum convolutional layers. TensorFlow/Keras enabled us to build and train classical models such as DenseNet121, as well as hybrid models that combine quantum and classical layers. Data management was handled with pandas, while OpenCV (cv2) and matplotlib supported image processing and visualization. The dataset was loaded using h5py, and scikit-learn facilitated train-test splitting and performance evaluation. Our hybrid quantum-classical model achieved a high accuracy of 94%, with detailed insights provided by the classification report and confusion matrix. This project demonstrates how quantum and deep learning technologies can work together to advance medical image classification.

Heart ECG Classification

In this project, we developed a highly accurate cardiovascular disease detection system by combining Deep Learning and Quantum Machine Learning (QML) to classify ECG images into four categories, including myocardial infarction and other abnormal heart conditions. The backbone of our model is EfficientNetB0, a lightweight yet powerful deep neural network that efficiently extracts meaningful features from ECG images. To boost performance, we integrated a Quantum Layer using Pennylane, which introduces quantum computation principles like superposition and entanglement to better capture complex data patterns. The hybrid architecture significantly improved both training speed and classification precision. With images resized to 64x64 pixels, the model achieved a weighted F1-score of 0.9917, while for high-resolution 255x255 images, it reached a near-perfect F1-score of 0.9986 with 100% accuracy on most classes. This approach demonstrates the power of combining quantum-enhanced learning with deep neural networks for fast, reliable, and scalable ECG-based heart disease detection.
